# Présentation du projet

   ℹ️Ce dépôt contient un projet SQL illustrant l’utilisation
**des commandes SQL fondamentales**(SELECT,SELECTDISTINCT,WHERE) ainsi qu’un
**workflow Gitsimple.**

#### 📂Structure du projet

```

sql-project/
     sql/
         commandes_fondamentales.sql

README.md
.gitignore
```
### 📥Récupération du projet

Le projet est récupéré àpartir d’undépôt distant à l’aide de la commande gitc clone:

```
git clone [Mon Projet GitHub](https://github.com/username/sql-project.git)
cd sql-project
```
 Cette étape permet d’obtenir une copie locale complète du projet.

 ### 🛢️Exécution du script SQL
Le fichier commandes fondamentales.sql contient plusieurs requtes permettant de :
-afficher toutes les données d’une table
-extraire des valeurs uniques
-filtrer les résultats selon une condition

Le script est exécuté dans un environnement SQL tel que :
-SQL Server Management Studio
-Azure Data Studio

### ✅Validation des résultats
Les résultats obtenus après l’exécution des requêtes permettent de vérifier
:
-la compréhension des commandes SQL fondamentales
-la cohérence des données retournées
   👍Conclusion Ce projet met en pratique les bases du langage SQL tout en
utilisant Git comme outil de gestion de versions pour récupérer et organiser
le travail.
